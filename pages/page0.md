
# 导航

将鼠标悬停在左下角以查看导航控制面板


|                                                     |                      |
| --------------------------------------------------- | -------------------------- |
| <kbd>→ 右箭头</kbd> / <kbd>空格键</kbd>                 | 下一个动画或下一张幻灯片     |
| <kbd>← 左箭头</kbd>  / <kbd>Shift</kbd>+<kbd>空格键</kbd> | 上一个动画或上一张幻灯片     |
| <kbd>↑ 上箭头</kbd>                                       | 上一张幻灯片              |
| <kbd>↓ 下箭头</kbd>                                     | 下一张幻灯片              |
